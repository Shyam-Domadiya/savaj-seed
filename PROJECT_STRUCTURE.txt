# Savaj Seeds - Project Overview

## 1. Technology Stack
This project is a modern Full-Stack Web Application built with:

*   **Framework**: Next.js 16 (App Router)
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS 4 (with OKLCH color variables for theming)
*   **UI Library**: Shadcn UI (Radix UI primitives + Tailwind)
*   **Icons**: Lucide React
*   **Animations**: Framer Motion, Tailwind Animate, and custom CSS keyframes
*   **Forms**: React Hook Form + Zod validation
*   **SEO**: Structured Data (JSON-LD) generation, Metadata API
*   **Analytics**: Vercel Analytics

## 2. Project Structure (`/app` Directory) (Next.js App Router)
The routing is file-system based within the `app` directory.

*   `app/layout.tsx`: The Root Layout. Contains global providers (Analytics, Toaster, Theme), fonts (Geist), and SEO metadata.
*   `app/page.tsx`: The Homepage. Features a hero section, feature highlights, seed categories, and testimonials.
*   `app/products/`: specific product listing and details pages.
*   `app/about/`: Company information page.
*   `app/contact/`: Contact form and location details.
*   `app/blog/`: Blog listing and post pages.
*   `app/certifications/`: Showcase of company certifications.
*   `app/locations/`: Dealer/store locator.
*   `app/search/`: Search functionality page.
*   `app/globals.css`: Global styles, including extensive custom animations and theme variables.

## 3. Key Components (`/components`)
Components are modular and categorized:

*   **`components/ui/`**: Base UI elements (Button, Card, Input, Sheet, etc.) from Shadcn UI.
*   **Functional Components**:
    *   `site-header.tsx` / `site-footer.tsx`: Global navigation and footer.
    *   `mobile-nav.tsx`: Responsive mobile menu.
    *   `product-*.tsx`: Components related to product display, filtering, and comparison.
    *   `testimonials-*.tsx`: Testimonial carousels and sections.
    *   `enhanced-contact-form.tsx`: Complex contact form with validation.
    *   `structured-data.tsx`: SEO component for injecting JSON-LD schema.
    *   `performance-monitor.tsx`: Utility for tracking app performance.

## 4. Key Features
*   **Advanced SEO**: Automatic sitemap generation (`sitemap.ts`), `robots.ts`, and rich snippet support via structured data.
*   **Performance Optimization**: Lazy loading images (`lazy-image.tsx`), progressive image loading (`progressive-image.tsx`), and resource prefetching.
*   **Accessibility**: `skip-nav.tsx`, `accessibility-controls.tsx`, `screen-reader-only.tsx`, and ARIA-compliant UI components.
*   **Theming**: Robust CSS variable system supporting light/dark modes with high-contrast options.
*   **Animations**: Custom "professional" animations defined in `globals.css` (fade-in, slide-up, float, etc.) used throughout the site.

## 5. Development Utilities
*   `lib/utils.ts`: Helper functions (like class merging).
*   `lib/seo.ts`: Schema generators for Business, Website, and Organization data.

This overview provides a roadmap of the existing codebase to assist with understanding or migration planning.
